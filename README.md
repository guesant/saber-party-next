# saber-party-next

## Dependências

- ffmpeg

## Instalação

Esse projeto utiliza o gerenciador de pacotes [pnpm](https://pnpm.js.org/).

```sh
npm i -g pnpm
git clone https://github.com/guesant/saber-party-next.git
cd saber-party-next
pnpm install
```

## Uso

```sh
# Dentro de saber-party-next
pnpx ts-node src/interfaces/cliDownloadLesson.ts
```
